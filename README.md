# Retail Sales Analysis

## 📌 Project Overview

This project analyzes retail sales data to identify trends, customer behavior, and business performance.

## 🎯 Objectives

* Analyze sales performance across regions
* Identify top-performing products
* Understand customer purchasing patterns

## 🛠 Tools Used

* Excel
* SQL
* Tableau
* Python

## 📊 Key Insights

* Top-performing categories contributed highest revenue
* Certain regions showed consistent growth
* Seasonal trends impacted sales significantly

## 📁 Project Structure

* data/ → raw dataset
* sql/ → SQL queries
* excel/ → Excel analysis
* tableau/ → dashboard
* Python/ → data analysis
